window 관련#

 Keys                                Description                               
 C-x o 
       
       		 witch cursor to another window. Try this now to switch between your    
Clojure file and the REPL.                                              
                                                                        
 C-x 1 
       
       
       		Delete all other windows, leaving only the current window in the frame. 
This doesn’t close your buffers, and it won’t cause you to lose any     
work.                                                                   
                                                                        
 C-x 2 
       		 Split frame above and below.                                           
                                                                        
 C-x 3 
       		 Split frame side by side.                                              
                                                                        
 C-x 0 
       		 Delete current window.

clojure 관련#

KeysDescription
C-c M-n M-nSwitch to namespace of current buffer.
C-x C-eEvaluate expression immediately preceding point.
C-c C-kCompile current buffer.
C-c C-d C-dDisplay documentation for symbol under point.
M-. and M-,Navigate to source code for symbol under point and return to your original buffer.
C-c C-d C-aApropros search; find arbitrary text across function names and documentation.
