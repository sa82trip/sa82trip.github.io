## 지금까지 블로그를 제대로 하지 않았던 이유? {#지금까지-블로그를-제대로-하지-않았던-이유}

1.  절대 핑계가 될 수 없다고 생각하지만, code를 블로그에 넣는게 조금이나마 불편했다고 느꼈고, 코딩을 공부하는데, 코드를 넣지 않고 블로그를 쓰려고 하는 생각을 갖게 되는 내 자신을 보고 이건 제대로 공부하는게 아니고, 써야 하니깐 쓰는거다 라는 생각이 들어서 그냥 이럴 빠엔 안한다 라고 생각했다.

    <!--more-->
2.  어떤 플랫폼이라도 내가 적은 자료가 나중에 내가 관리하기에 편하지 않다는 생각이 들었다.
    -   노션은 백업을 하면 다시 완벽하게 복구 되지 않았고,
    -   여타 블로그들은 백업 하는 거 자체가 뭔가 복잡하게 느껴졌었다.


## 이 블로그를 시작하게 된 이유 {#이-블로그를-시작하게-된-이유}

-   기술적인 생각을 정리하려는 블로그를 운영하려고 마음 먹은지 꽤나 오래되었다. 그래서 이번에야 말로 혼자 공부하는 것도 좀 보기 좋게 기록으로 남겨 놓고, 내 머리속에도 잘 정리 시켜서 넣자는 생각으로 요번엔 의지를 강하게 먹었다.
-   우연찮게 emacs를 사용해보고 싶게 되어서 공부를 하다 org mode를 알게 되고 조금이나마 익숙해져갈 때쯤 ox-hugo를 이용하면 org file을 바로 hugo용 markdown file로 쉽게 export된다는 사실에 code를 넣기도 편하고, 내가 적은 정보를 내가 관리하기도 쉽다면 드디어 좋은 방법을 찾았다라는 생각에 블로그를 시작하자고 마음 먹었음
-   그리고 org파일을 저장하면 live preview를 볼 수 있다.
    ```org { linenos=true, linenostart=1 }

          # Local Variables:
          # eval : (org-hugo-auto-export-mode)
          # End:
    ```
